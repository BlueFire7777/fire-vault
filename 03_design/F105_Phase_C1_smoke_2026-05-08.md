## 8. 走行中の発見 + 修正 (smoke で発覚)

★ **bug**: code 4 桁入力 / response 5 桁不整合で派生 0 件
- 初回 smoke 走行で 5 分 / 15 分派生が **0 件** と判明
- 原因: J-Quants は code='7203' 入力に対し response Code='72030' (5 桁) を返す。
  fetch_one が compute_vwap / derive_intraday_bars に 4 桁 code を渡し、
  DB 内 5 桁 code と SQL 不一致で 0 row 返却
- 修正 (commit 8d66c14):
  - `_normalize_code()` helper 追加 (4 桁→5 桁)
  - fetch_one で `saved_code = rows[0]["Code"]` を以降に渡す
  - run_backfill で `_normalize_code(code)` を resume 確認に使う
- 修正後 smoke 再走行: 5 分 401 / 15 分 144 row、7203 5/1 で 5 分 67 / 15 分 24
  ✅ HQ 期待値一致
- test 18 PASS 維持
