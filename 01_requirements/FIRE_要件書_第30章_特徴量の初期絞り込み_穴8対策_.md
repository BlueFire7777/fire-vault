# 第30章: 特徴量の初期絞り込み(穴8対策)

## 穴8の本質

再現性を高めようとすると特徴量を全部入れたくなる。しかし特徴量を増やしすぎると**「類似局面が見つからない・パターン件数が細切れになる・過剰適合する」**危険がある。

## 初期特徴量セット(最小コアセット)

最初は以下の**6系統**に絞る。

| 系統 | 内容 |
|---|---|
| 材料 | 材料種別(好決算・上方修正・自社株買い等)/ 材料強度 / 材料の新規性 |
| 地合い | 日経225方向 / TOPIX方向 / セクター強弱 / レジーム分類 |
| 出来高 | 異常出来高倍率 / 出来高の方向性 |
| 値動き形状 | ギャップ方向・大きさ / 前場の値動きパターン / 高値更新・安値更新 |
| 時間帯 | エントリー時間帯(寄り・前場前半・前場後半等) |
| 執行品質 | スプレッド / 板の厚さ / 約定品質スコア |

**方針**: テクニカル指標の細かい値・ファンダメンタル詳細・板の微小な歪みは**後から追加**する。最初は「類似局面が十分に見つかる粒度」を優先する。

## 本章の要件ID

- **R-30-01** 初期特徴量6系統
- **R-30-02** 細かいテクニカル・ファンダ詳細は後から追加
