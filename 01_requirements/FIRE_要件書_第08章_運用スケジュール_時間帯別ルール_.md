# 第08章: 運用スケジュール(時間帯別ルール)

## 時間帯別ルール

| 時間帯 | フェーズ | 内容 |
|---|---|---|
| 8:00〜8:25 | 朝分析 | 市況・ニュース・セクター・朝候補抽出・ポートフォリオ偏り確認 |
| 8:25〜8:30 | 朝レポート送信 | PDF生成、LINE送信 |
| 8:30〜8:40 | 実行候補絞り込み | 朝の5候補からデイトレ1〜3銘柄へ厳選 |
| 8:40 | 候補通知 | 本日のデイトレ・スイング・長期候補をLINE送信(戦略コメント付き) |
| 9:00〜9:08 | 寄り付き観察 | 飛びつき禁止、GU/GD確認、初動ノイズ警戒 |
| 9:08〜9:20 | 初動判断 | 本命候補のみ監視、条件成立なら通知 |
| 9:20〜10:30 | 前場メイン | デイトレ主戦場、新規エントリー最優先 |
| 10:30〜11:00 | 前場後半整理 | 伸びない銘柄の整理、後場候補仮決定 |
| 11:00〜12:20 | 昼休み分析 | 前場レビュー、後場本命再設定 |
| 12:20〜12:35 | 後場寄り監視 | GU/GD確認、重要価格帯の再確認 |
| 12:35〜14:00 | 後場メイン | 既存ポジション管理優先、新規は慎重 |
| 14:00〜14:45 | クローズ準備 | 信用建玉整理優先、新規は原則終了 |
| 14:45〜15:10 | 強制クローズ | 信用建玉を必ず整理、未約定時は強制クローズ手順へ |
| 15:10〜15:30 | 引け確認 | 現物持越し判断、当日集計、翌営業日候補仮保存 |
| 16:00 | 引け後レビュー送信 | PDF生成、LINE送信 |

## 本章の要件ID

- **R-08-01** 8:00-8:25 朝分析
- **R-08-02** 8:25-8:30 朝レポート送信
- **R-08-03** 8:30-8:40 実行候補絞り込み
- **R-08-04** 8:40 候補通知
- **R-08-05** 9:00-9:08 寄り付き観察(飛びつき禁止)
- **R-08-06** 9:08-9:20 初動判断
- **R-08-07** 9:20-10:30 前場メイン
- **R-08-08** 10:30-11:00 前場後半整理
- **R-08-09** 11:00-12:20 昼休み分析
- **R-08-10** 12:20-12:35 後場寄り監視
- **R-08-11** 12:35-14:00 後場メイン
- **R-08-12** 14:00-14:45 クローズ準備
- **R-08-13** 14:45-15:10 強制クローズ
- **R-08-14** 15:10-15:30 引け確認
- **R-08-15** 16:00 引け後レビュー送信
