# F051: 仮想建玉計算 / 平均建値 / 評価損益 / 実現損益 / 余力管理

## 概要

F050 (Paper Live) で記録される仮想エントリー / 利確 / 損切イベントから、仮想建玉 (positions) を構築し、平均建値 / 評価損益 / 実現損益 / 余力を計算する。

## スコープ

- 仮想建玉テーブル (paper_live_positions) の追加
- 平均建値の加重平均計算
- 評価損益 (含み損益) の即時計算
- 実現損益 (クローズ時) の確定
- 余力管理 (使用中証拠金 / 残余力)
