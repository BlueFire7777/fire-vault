# F201: 頻度監視

## 概要

`patterns/frequency.py` 新規作成。日次・週次トレード頻度集計と 2 週連続不足検知。

## 実装内容

### 主要モジュール

- `patterns/frequency.py`
  - dataclass: `DailyFrequencyReport` / `WeeklyFrequencyReport`
  - `get_daily_report` / `get_weekly_report`: 日次・週次頻度集計 (R-25-04)
  - `detect_frequency_shortage`: 2 週連続不足検知 (R-25-05)
  - `get_dashboard_metrics`: Dashboard 表示用集計 (R-25-06)
- CLI: `python -m patterns.frequency --mode shortage`

### 定数

- `DAILY_TARGET = (0, 3)`
- `WEEKLY_TARGET = (5, 10)`

### キーポイント

- マイグレーション不要: F200 の `lane` / `strategy_type` カラムを利用
- F181 同時リファクタ: `StrategyType` を `lanes.py` に統合し重複解消
